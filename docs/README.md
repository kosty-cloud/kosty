## 🚀 Quick Start

```bash
# Install
git clone https://github.com/yassirkachri/kosty.git
cd kosty && ./install.sh

# Configure AWS
aws configure

# Run audit
kosty --all --output json

# View dashboard
open dashboard/index.html
```
